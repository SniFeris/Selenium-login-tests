# Selenium Login Test Automation Project

Small QA automation project for practicing Selenium WebDriver, pytest and page object model.

-Valid login
-Invalid login with wrong password
-Invalid login with empty username
-Invalid login with empty password
-Invalid login with empty username and empty password
-Logout
-Back button behaviour after logout
-Login page visibility
-Password input type validation

## Project structure

```text
selenium project/

    test_login.py
    login_page.py
    README.md
  
```
## How to run tests

```bash
py -m pytest selenium_project/test_login.py
```
